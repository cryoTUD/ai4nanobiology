# AI4Nanobiology: Setting things up in DelftBlue
### Step 1: Getting access

1. You can request access for all students by raising a topdesk call [here](https://tudelft.topdesk.net/tas/public/ssp/content/serviceflow?unid=58cebc4f3e9b4e67b87ddd2b54bf1666)
a) Purpose: Students to access DelftBlue for the duration of a master course (NB4170) in Q4
b) Resources: Access to gpu-a100-small (or gpu-a100) and compute nodes. Compute nodes used for interactive jupyter notebook execution. GPUs for training a neural network using python scripts
c) Student list: Export from Brightspace. Go to Course Admin > Classlist > Select all > Export as CSV
Upload the student list along with the request. All the students who are in the list will automatically get access to DelftBlue. 

Special note: External students will need to go through some extra step (same holds true for instructors, and TA's from outside TU Delft). 

a) You should raise [generic topdesk request](https://tudelft.topdesk.net/tas/public/ssp/content/serviceflow?unid=d4bd7e8e403b44feb221daa17d1aad61) with the following text: 
```bash
Please add NetID xxxxxxx to AD group Apps-OpenVPN-UD-HPCGuest
```
